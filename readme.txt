欢迎使用JSPGOU 店中店版，请按照以下步骤进行软件安装：

一、安装运行环境（JDK8+TOMCAT8+MYSQL5.6及以上);

二、将程序解压后的jspgou.sql导入到数据中，修改数据库连接信息
\WEB-INF\classes\application-prod.properties 

三、将程序解压后的ROOT文件夹拷贝到tomcat安装目录下的webapps文件夹下（例如：D:\Tomcat8\webapps\），启动tomcat

四、在浏览器的地址栏中输入http://localhost:8080，若系统正常显示网站首页则表示本系统顺利安装完成。

五、自带数据库中短信及邮件配置参数已重置为随机参数，如需要体验短信或邮件相关功能，请自行前往第三方服务商获取相关配置参数
